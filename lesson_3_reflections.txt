When would you want to create remote repository rather than keeping all your 
work local?
When I want to share my work with others

Why might you want to always pull changes manually rather than having Git 
automatically stay up-to-date with your remote repository?
Porque puede que mi copia local este mas avanzada que el remoto y no quiero 
perder mi avance

Describe the differences between forks, clones, and branches. When would you use
one instead of another?
Forks -> Copiar un repo de github a github
Clone -> Copia un repo de github a local, o de local a local
Branches -> Versiones parelelas de un mismo repo.
